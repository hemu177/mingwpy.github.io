############################
README for mingwpy.github.io
############################

This is the repository housing the static HTML and Sphinx_ sources for the
mingwpy web pages.  Find the built static HTML in the ``master`` branch.

The sources are Sphinx pages written in reStructuredText_.  Find the sources
in the ``source`` branch.

Build the sources into static HTML pages and push the built pages to github
with ``make github``.

.. _sphinx: http://sphinx-doc.org
.. _restructuredtext: http://docutils.sourceforge.net/rst.html
